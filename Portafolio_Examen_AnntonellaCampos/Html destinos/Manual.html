Manual

Este es un manual creados para ayudar a diseñadores nuevas en todo el proceso que con lleva imprimir sus creaciones. Creado para la asignatura de Pre-prensa, este fue un proyecto en pareja. El proceso incluyo investigación, bocetaje, pruebas y creación final del manual que consistió en una caja que incluye el manual, y tarjetas adicionales para ayudar a los diseñadores.
